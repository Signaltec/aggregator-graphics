# Прототипирование графиков статистики агрегатора

## Установка зависимостей

Будет установлена InfluxDB 0.9.5 и npm-пакеты

```bash
./install.sh
```
## Старт. Запустите гульп и откройте в браузере localhost:3000

```bash
gulp
```
